# ESP32 PlatformIO Project

## Thông tin
- Sinh viên: Trần Quang Anh  
- MSSV: B22DCDT025  
- Board: ESP32  
- Framework: Arduino (PlatformIO)  
- Cảm biến: DHT11  

## Mô tả
- Đọc nhiệt độ và độ ẩm từ cảm biến DHT11 và hiển thị qua Serial Monitor.(one wire)
- Đọc nhiệt độ và độ ẩm từ cảm biến DHT11 và hiển thị qua màn hình LCD.(i2c)
- Đọc nhiệt độ và độ ẩm từ cảm biến gia tốc và hiển thị qua màn hình TFT.(spi)
